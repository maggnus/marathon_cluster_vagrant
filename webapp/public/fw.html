Reference for v1.3.6

The reference doc tells you about the components included on the latest version of Siimple, and how to use them. For checking how to download and install Siimple, read the getting started guide.

Siimple provides a small, light, responsive and clean framework, build in ~250 lines of code. Also, the framework comes with a ~6KB minimized version.

TYPOGRAPHY

Siimple uses Open Sans as the default font. The default font-size is 16, and the default font-weight is 300 (light). Siimple comes with bold tag, italic, underline, links and small text . The default text color is #526475, and the emphasis color is #09a0f6.

BUTTONS

Add a .btn class to transform your div, a or button element into a normal button, or add the .btn-outline class to transform your element to an outline button. You can change the button size by adding a .btn-small or .btn-big class.

<!-- Normal buttons -->
<div class="btn btn-small">Small button</div>
<div class="btn">Default button</div>
<div class="btn btn-big">Big button</div>

<!-- Outline buttons -->
<div class="btn-outline btn-small">Small button</div>
<div class="btn-outline">Default button</div>
<div class="btn-outline btn-big">Big button</div>

You can change the button color by adding a new class to your button with the next style:

<!-- For change the normal button color -->
.btn-color { background-color: [color]; }

<!-- For change the outline button color -->
.btn-outline-color { border-color: [color]; color: [color]; }
.btn-outline-color:hover { background-color: [color]; color: #ffffff; }

FORMS

Add a class .form-input to your text-based input fields (text, password and submit types), .form-select to your select fields, and .form-textarea to your textarea. By default, the width of all form elements is 100%, but you can adjust the width to the content adding the .form-auto class.

<!-- Input text -->
<input type="text" class="form-input" placeholder="Your email">

<!-- Select -->
<select class="form-select">
  <option>Male</option>
  <option>Female</option>
</select>

<!-- Textarea -->
<textarea class="form-textarea"></textarea>

<!-- Submit -->
<input type="submit" class="form-input" value="Submit form">

ALERTS

You can create a blue alert adding the .alert class to your div. The default color for the alert is blue, but you can change it by adding .alert-[type] class to the alert, where [type] can be done, warning or error.

This is an alert, and this is a link inside the alert.
This is a done alert.
This is a warning alert.
This is an error alert.
<div class="alert">This is an alert.</div>
<div class="alert alert-done">This is a done alert.</div>
<div class="alert alert-warning">This is a warning alert.</div>
<div class="alert alert-error">This is an error alert.</div>

TABLES

Simply add .table to the class of your table. You can add a row into a <thead>, and this row will be the head of tour table. Tables always comes with full width property.

GRID
